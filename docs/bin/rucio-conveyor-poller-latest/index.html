Running rucio-conveyor-poller-latest

usage: rucio-conveyor-poller-latest [-h] [--run-once]
                                    [--total-threads TOTAL_THREADS]
                                    [--last-nhours LAST_NHOURS]
                                    [--fts-wait FTS_WAIT]
                                    [--external-hosts EXTERNAL_HOSTS [EXTERNAL_HOSTS ...]]
                                    [--allow]

optional arguments:
  -h, --help            show this help message and exit
  --run-once            One iteration only
  --total-threads TOTAL_THREADS
                        Concurrency control: total number of threads per
                        process
  --last-nhours LAST_NHOURS
                        Poll control: poll FTS jobs in last n hours with given
                        state in "latest" mode
  --fts-wait FTS_WAIT   Poll control: wait seconds to poll FTS server in
                        "latest" mode
  --external-hosts EXTERNAL_HOSTS [EXTERNAL_HOSTS ...]
                        Poll control: List of FTS hosts
  --allow               This daemon is deprecated and will be removed.
                        Explicitly turn it on.
